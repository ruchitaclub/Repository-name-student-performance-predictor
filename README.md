## 📌 Project Overview

This is a **full-stack Machine Learning web application** that predicts student performance based on study hours and attendance.

## 🔥 Features

- 🔐 Login & Registration System
- 👥 Multi-user support
- 🤖 Machine Learning Prediction
- 📊 Interactive Graph Visualization
- 📁 Save & Download Prediction History
- 🌐 Deployed Online (Streamlit)
